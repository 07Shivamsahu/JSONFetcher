# JSON Fetcher

This JSON Fetcher is a simple web application tool designed to display data fetched from JSON files and render them on a webpage.
## Features

- Fetch data from multiple JSON files.
- Render Json on the webpage.
- Display question numbers, text, choices, short descriptions, and solutions.

## Technology used

- HTML
- JavaScript
- JSON
- Node.js server 

## Getting Started

To get started with the JSON Fetcher tool, follow these steps:

1. Clone this repository to your local machine:

   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```
   cd project-directory
   ```

3. Start a local server:

   You can use any local server of your choice. For example, if you have Node.js installed, you can use the `http-server` package:

   ```
   npm install -g http-server
   ```

   Then, start the server:

   ```
   http-server
   ```

4. Open your web browser and navigate to the local server's URL (usually `http://localhost:8080`).

5. You should see the tool displaying the data fetched from the JSON files.
